# Pantry Pilot Visual Identity Style Guide

## 1. Color Palette
- **Primary (Action):** Sage Green (`#8DAA81`)
  - Usage: "Add," "Save," and "Confirm" buttons.
- **Secondary (Navigation):** Sky Blue (`#5DADE2`)
  - Usage: Active tab states, links, and the paper airplane "Move" icon.
- **Accent (Natural):** Warm Oak (`#B08D57`)
  - Usage: Pantry-related headers or subtle borders.
- **Background:** Off-White (`#F9F9F9`)
  - Usage: Main app background.
- **Text:** Charcoal Gray (`#333333`)
  - Usage: Primary text color (avoid pure black).

## 2. Typography & Shapes
- **Font:** Inter (Sans-serif)
- **Corner Radius:** 12px (`rounded-xl`)
  - Applied to: Buttons, input fields, and storage cards.
- **Shadows:** Soft Depth
  - Subtle, blurry shadows on cards to create a floating effect.

## 3. Layout Constraints
- **Mobile-First:** Single-column layout for phones.
- **Card-Based:** Storage units represented as white cards.
- **Spacing:** Generous white space (16px padding).
